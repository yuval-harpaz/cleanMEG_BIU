cleaning MEG data at Bar-Ilan University with matlab tools.

Hi. here you can find the latest version of our cleaning procedures. it includes the pdf4D tool. The main function 'createCleanFile.m' was written by Prof. Abeles, and was published with Idan Tal. Please refer to this work (Tal & Abeles, 2013) if you use the function. createCleanFile works on 4D data only. You can use correctLF and correctHB to clean line frequency (50Hz) and heartbeat artifacts for any MEG / EEG data. to clean heartbeats, try: clean=correctHB(data,samplingRate); Here are some explanations and examples for how it works, input, output and plots of the function.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/yuval-harpaz/cleanMEG_BIU

Install

Add cleanMEG_BIU/cleanMEG with subfolders to your matlab path. use cleanFileExample.m to call the function.

Examples

BIU users, use cleaneFileExample.m for createCleanFile. BIU users who want to use correctHB see the 'BIU users section in correctHB_4D_example.

There are also example scripts in the repository for CTF, Elekta and 4D users (not from BIU) showing how to clean HB or LF.
